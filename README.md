# はてなブログ投稿用 Obsidian プラグイン

Obsidianノートをはてなブログに投稿するプラグインです。

## 設定

はてなブログへの投稿には、はてなAPIを利用します。以下の手順で情報を取得し、設定画面に入力してください。

1. はてなブログの設定画面を開く
2. Advanced > AtomPubの設定を開く
3. Root endpointをコピーする
4. アカウント設定画面を開く
5. API keyをコピーする

## 使い方

### コマンド

- `Hatena: Publish this note`: 現在開いているノートをはてなブログに投稿します
